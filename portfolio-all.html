GitFit

We created a site called "GitFit". A fun, and pun filled application that helps coders/users log workouts, set personal fitness records, and join fitness challenges. Our site allows password protected users to have all their information private and available to them. Written in Javascript, using Node.js, Express.js, PostgreSQL. To assess our database's functionality, we used BeeKeeper. In this project I was only hands on in the backend development. While I enjoy front-end development the most, I pushed and succeed in my growth of backend development, especially with the databases.

  • Date:- October 2020
  • GitHub- https://github.com/plooney81/Backend-Project
  • Architects:- Alison Manning, Pete Looney, Jacky Cheung

Catch Charlie

Catch Chalie is a "cat and mouse" themed video game written in Python. To aid in animation and functionality, the import "pygame" was installed. This project refined my skills in classes, if and while statements, and enabling key movements. The greatest challenge in this project was the use of animation. Both Charlie, (the dog), and the vehicles are set to move randomly. While the catcher, (a photo of myself), is controlled by the keyboard. This project was not only widly fun, it was personal. My sweet dog passed this year due to heart failure, and it was nice to remember him for what he loved most, which was running off leash.

  • Date:- September 2020
  • GitHub- https://github.com/alisonlauren/Catch-Charlie
  • Architect:- Alison Manning

What's Your Flavorite?

"What's Your Flavorite?" is a web application written in HTML, CSS and Javascript. This application will allow users to input the grocery items they have onhand and recieve an output of the recipes they can make. The video you see here is highlighting one of the pages where the user get's to "Choose their Flavorites" grocery items. With the use of checkboxes, forms and adding the users data into local storage, our site runs effectively and user-friendly. My part of the project was design in HTML & CSS. The biggest challenge was making sure the webiste was responsive. Not only am I comfortable with creating responsivity, I understand it's important in this mobile-age.

  • Date:- September 2020
  • GitHub- https://github.com/alessandraburckhalter/Whats-Your-Flavorite
  • Architects:- Alison Manning, Alessandra Burckhalter, Thor Hlavaty, and Heeyoung Song
